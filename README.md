# 🎬 MovieFinder Client

This is the frontend client of the **MovieFinder App** built with React, TypeScript, and Vite.

## 🚀 Features
- 🔍 Search for movies in real-time
- 🎞️ View movie details in a modal
- ⏳ Debounced search input
- 📄 Pagination support

## 🛠️ Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite

## 📦 Installation

```bash
npm install
npm run dev
